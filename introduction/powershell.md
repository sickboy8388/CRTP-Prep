# Powershell - Introduzione

* Powershell (PS) e' una soluzione cross-platform per l'automazione di task, CLI based, con uno scripting language
* Installato di default su tutti i moderni OS
* PowerShell non e' poweshell.exe, ma System.Management.Automation.dll
* Utilizzeremo Windows PowerShell, c'e' una piattaforma indipendente inoltre, PowerShell Core
