# Policy Dominio

#### Ottenere policy dell'attuale dominio

```powershell
Get-DomainPolicyData (PowerView)
```

```powershell
(Get-DomainPolicyData).systemaccess (ActiveDirectory Module)
```

#### Ottenere policy di un altro dominio

```powershell
(Get-DomainPolicyData).systemaccess (ActiveDirectory Module)
```
